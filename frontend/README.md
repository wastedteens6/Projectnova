# ProjectNova - Frontend

React + Vite frontend application for ProjectNova.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Create `.env.local`:

```
VITE_API_URL=http://localhost:5000
```

## Features

- Landing page with hero section and featured projects
- Pricing page with 4 tier options
- Projects catalog with filtering
- Project detail pages with tier selection
- User authentication (login/register)
- Dashboard for user account management
- Support ticket submission
- Responsive design with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Axios
- React Router
